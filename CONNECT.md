# Как подключить друга

## Вариант 1: Локальная сеть (самый простой)
Если вы в одной WiFi сети:

1. Узнай свой IP адрес:
   - Открой командную строку (Win+R → cmd)
   - Введи: `ipconfig`
   - Найди "IPv4-адрес" (например: 192.168.1.100)

2. Запусти сервер на своем компе
3. Друг открывает в браузере: `http://ТВОЙ_IP:3000` (например: http://192.168.1.100:3000)
4. Вводите один и тот же код комнаты

## Вариант 2: Через интернет (ngrok - бесплатно)

1. Скачай ngrok: https://ngrok.com/download
2. Зарегистрируйся и получи токен
3. Запусти сервер (start.bat)
4. В другом терминале запусти: `ngrok http 3000`
5. Ngrok даст тебе ссылку типа: `https://abc123.ngrok.io`
6. Отправь эту ссылку другу
7. Вводите один и тот же код комнаты

## Вариант 3: Проброс портов на роутере (сложнее)

1. Зайди в настройки роутера (обычно 192.168.1.1)
2. Найди "Port Forwarding" или "Проброс портов"
3. Пробрось порт 3000 на свой локальный IP
4. Узнай свой внешний IP: https://2ip.ru
5. Друг открывает: `http://ТВОЙ_ВНЕШНИЙ_IP:3000`

## Как использовать:

1. Ты создаешь комнату (получаешь код, например: ABC123)
2. Выбираешь источник видео (YouTube/ссылка/файл)
3. Отправляешь другу:
   - Ссылку на сервер
   - Код комнаты
4. Друг заходит по ссылке, вводит код комнаты
5. Видео синхронизируется автоматически!

**Важно:** Для варианта с файлом - файл должен быть у каждого, или используй YouTube/ссылку на видео в интернете.
