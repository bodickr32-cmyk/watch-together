# Watch Together - Совместный просмотр видео

Веб-приложение для совместного просмотра видео с друзьями.

## Функции

- 🎬 Синхронный просмотр видео
- 💬 Чат в реальном времени
- 🔗 Простые коды комнат
- 🌐 Работает в любом браузере
- 📺 YouTube, прямые ссылки, локальные файлы

## Быстрый старт (локально)

1. Запусти `start.bat` (двойной клик)
2. Открой браузер: http://localhost:3000
3. Создай комнату, выбери видео
4. Поделись кодом с друзьями

## Залить на бесплатный хостинг

Смотри `DEPLOY.md` - там пошаговая инструкция как залить на Render/Railway/Glitch бесплатно.

После деплоя получишь постоянную ссылку типа:
- `https://watch-together.onrender.com`
- `https://watch-together.up.railway.app`

Просто отправляй эту ссылку друзьям - никаких ngrok и проброса портов!

## Технологии

- Node.js + Express
- Socket.io для синхронизации
- Чистый HTML/CSS/JavaScript
